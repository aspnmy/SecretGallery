# GoComicMosaic 前端项目改造计划

## 一、改造目标

1. 将 Vue 3 项目迁移到 React 18 + Next.js
2. 实现瀑布流图片布局
3. 集成图片压缩功能
4. 优化视频播放体验
5. 增强搜索功能
6. 支持主题切换
7. 保持与后端 API 兼容

## 二、技术栈

- **前端框架**：React 18 + Next.js
- **状态管理**：Zustand
- **网络请求**：Axios
- **图片处理**：compressorjs
- **瀑布流布局**：react-photo-album
- **UI 组件**：自定义组件 + CSS 变量
- **构建工具**：Next.js 内置
